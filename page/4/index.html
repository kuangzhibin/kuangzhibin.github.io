Hello World

我是一名程序员，更准确的说是前端（F2E）。我有必要在这里输出“Hello world!”作为序幕。
这个Blog是基于“Hexo+Github”的静态网站。
初衷我是奔着 Nodejs 来的，机缘巧合下看到了一篇名为”Hexo 在 github 上构建免费的 Web 应用“，便心动了！
尤其喜欢的是Hexo支持Markdown（一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转化成有效的XHTML（或者HTML）文档。）
